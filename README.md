#  Phishing Email Simulator

##  Описание
**Phishing Email Simulator** - это консольная утилита на Python для генерации шаблонов фишинговых писем. Проект создан исключительно в образовательных целях для демонстрации методов социальной инженерии и обучения сотрудников основам кибербезопасности (Security Awareness).

Инструмент позволяет автоматизировать создание правдоподобных писем, подставляя данные жертвы в заранее подготовленные сценарии.

##  Возможности
* **Интерактивное CLI-меню:** Удобный интерфейс с защитой от некорректного ввода пользователя.
* **Динамическая генерация текста:** Использование `f-strings` для автоматической подстановки имени отправителя, email-адреса жертвы и фишинговых ссылок.
* **Реалистичные сценарии:** Включает 4 классических шаблона социальной инженерии:
  1. Сброс пароля (Срочность)
  2. Поддельный счет/Invoice (Финансовый триггер)
  3. Предупреждение о безопасности (Страх взлома)
  4. Обновление политик (Корпоративная рутина)
* **Встроенный модуль `datetime`:** Автоматическая генерация текущей даты и времени для усиления эффекта срочности и правдоподобности.

##  Установка и запуск
1. Убедитесь, что у вас установлен **Python 3.x**.
2. Склонируйте репозиторий:
  ```bash
  git clone [git@github.com:aibraikhan/Phishing-Email-Simulator.git](https://github.com/aibraikhan/Phishing-Email-Simulator.git)
  ```
3. Перейдите в папку проекта и запустите скрипт:
  ```bash
  python phishing_email_sim.py
  ```
4. Следуйте инструкциям на экране для выбора шаблона и ввода данных.
